Documentation for UVM-Connect comes in 3 forms,

- Text
- HTML
- .pdf "wholebook"

For text please see,

    ./docs/OVERVIEW.txt (and other *.txt files)

For HTML point your HTML browser to,

    ./docs/html/index.html

or, alternatively, you can try the Verification Academy location,

    https://verificationacademy.com/verification-methodology-reference/uvmc-2.3.0/docs/html/index.html

For the "wholebook" .pdf form see,

    ./docs/UvmConnectPrimer.pdf

In particular in any of the 3 forms, for the fastest way to get started with
using UVM-Connect see please the section entitled,

    "Quickstart 1-2-3 to running examples"
